# AdventureWorks Business Intelligence Dashboard

An end-to-end Business Intelligence project built using **Power BI** and **PostgreSQL** to analyze AdventureWorks sales data and transform raw data into interactive dashboards that support business decision-making.

---

## Project Overview

This project demonstrates the complete Business Intelligence workflow, from database creation and SQL data preparation to dashboard development and business insights generation.

The dashboard provides an interactive analysis of:

- Executive Overview
- Sales Analysis
- Customer Analysis
- Product Analysis
- Geographical Analysis
- Returns Analysis
- Business Insights & Recommendations

## Dashboard Pages

### Executive Overview
Provides a high-level summary of business performance using KPIs, revenue trends, and category insights.

### Sales Analysis
Analyzes sales trends, monthly performance, order patterns, and revenue distribution.

### Customer Analysis
Explores customer demographics, purchasing behavior, customer segments, and revenue contribution.

### Product Analysis
Evaluates product performance, category-wise sales, top and bottom-performing products, and product trends.

### Geographical Analysis
Analyzes revenue and order distribution across countries and sales territories.

### Returns Analysis
Monitors product returns, return trends, return rate, and identifies products with the highest returns.

### Business Insights & Recommendations
Summarizes key findings, growth opportunities, business risks, and strategic recommendations based on the analysis.
